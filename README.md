# Capture The Flag Intro

A simple website hosting introductory Capture The Flag challenges.

## How to Run

1. Make sure you have Node.js installed.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the server.
4. Open your browser and go to `http://localhost:3000`.

## Challenges

1. **View Source**: Find the flag in the HTML source code.
2. **Math Time**: Solve a simple math problem.
3. **Riddle**: Answer a riddle.
4. **Hidden Message**: Decode a base64 encoded message.
